## Upload JSON

Load a session by uploading a JSON file.

<ol class="docs-steps">
  <li>
    <span class="docs-step-num">1</span>
    <div>
      <strong>Click "Upload"</strong>
      <p>Find the Upload button in the top navigation bar.</p>
    </div>
  </li>
  <li>
    <span class="docs-step-num">2</span>
    <div>
      <strong>Select your JSON file</strong>
      <p>Choose a session file in OpenCode JSON format.</p>
    </div>
  </li>
  <li>
    <span class="docs-step-num">3</span>
    <div>
      <strong>Explore the dashboard</strong>
      <p>The Overview, Timeline, Messages, Tools, and Insights tabs will populate automatically.</p>
    </div>
  </li>
</ol>
